# 🏆 Sport Management System

Une application web complète et sécurisée pour la gestion d'entités sportives, conçue pour centraliser les données des équipes, des joueurs et des stades.

## 🚀 Fonctionnalités Clés

* **Gestion Complète (CRUD) :** Ajout, modification et suppression d'équipes, de joueurs et de stades.
* **Interface Intuitive :** Développée avec Angular pour une expérience utilisateur fluide et réactive.
* **Authentification Sécurisée :** Système d'inscription et de connexion robuste utilisant **JWT (JSON Web Tokens)**.
* **API RESTful :** Backend performant assurant une communication rapide entre le serveur et le client.

## 🛠 Tech Stack

* **Frontend :** Angular
* **Backend :** Node.js & Express.js
* **Base de données :** MongoDB
* **Sécurité :** JWT (Authentication & Authorization)

## 📋 Prérequis

* Node.js (v14+)
* MongoDB
* Angular CLI
